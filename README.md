# 🎬 CineMatch — Movie Recommendation System

A content-based movie recommender that suggests similar films using cosine similarity.

## Features
- 🔍 Search from 5000+ movies
- 🎯 Content-based filtering
- 🖼️ Movie posters via OMDb API
- ✨ Cinematic UI

## Tech Stack
- Python, Streamlit, Pandas, Scikit-learn
